# nepali-hindi-classifier
HuggingFace NLP Model to classify if a word is Hindi or Nepali in Devanagari
हगींगफेसमा लेखीएको प्राक्रितीक भाषा नमुना जसले एक शब्द नेपाली हो की हिंदी छुट्टयाँउछ।

## How to use/ कसरी चलाउने?:
Usage is located in the using.py file.
using.py नामक फाइलमा यो नमुना चलाउने निर्देसन् छन्।

## Where can I see the model and make changes to it?
The model is located in https://huggingface.co/Emalper/hindi-nepali-token-classification and you can fine tune it if you like for any task you want.
https://huggingface.co/Emalper/hindi-nepali-token-classification मा यो नमुनाको आन्तरीक तंत्र राखीएको छ। यसको प्रयोग गरेर यसलाइ अझै अडीग बनाउन नहीच्कीचाउनू होला।
